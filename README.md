# YKS Puanım Gizlilik Politikası

Bu klasörde Google Play ve App Store için kullanılacak `privacy-policy.html` dosyası bulunur.

Önerilen yayın yöntemi:
1. GitHub'da `yks-puanim` adlı public repo oluştur.
2. `privacy-policy.html` dosyasını repo köküne yükle.
3. Repo Settings > Pages bölümünden GitHub Pages'i aktif et.
4. Privacy Policy URL olarak oluşan public linki kullan.

Örnek URL:
https://atazegullu.github.io/yks-puanim/privacy-policy.html
